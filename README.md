# 📊 Big Data Analytics I: Modelagem e Análise Estatística (BDN-003)

Bem-vindo ao meu repositório de estudos da disciplina **BDN-003**. Aqui documento a minha evolução na aplicação de métodos matemáticos e modelos estatísticos para mineração e análise de dados, utilizando a linguagem **R**.

## 🎯 Objetivos de Aprendizagem
O foco deste repositório é demonstrar a minha capacidade em:
* Executar **Análise Exploratória de Dados (EDA)** completa.
* Construir sistemas de visualização de dados para dimensões superiores.
* Implementar técnicas estatísticas multivariadas para extração de *insights*.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** R
* **Ambiente:** RStudio / Pop!_OS
* **Bibliotecas Principais:** `ggplot2`, `dplyr`, `tidyr`, `plotly`, `corrplot`
* **Base de Dados:** Importação, limpeza e gestão de estruturas de dados.

## 📖 Conteúdo Desenvolvido

### 💻 Programação e Gestão de Dados
* Introdução ao ecossistema R e gestão de pacotes.
* Manipulação de estruturas de dados (Vectors, Matrices, Data Frames, Lists).
* Técnicas de importação e saneamento de bases de dados.

### 📈 Visualização de Dados (Data Viz)
Nesta seção, exploro diferentes formas de representar dados visualmente:
* **Distribuição:** Histogramas, Gráficos de Densidade e Box Plots.
* **Relacionamento:** Scatter Plots (Dispersão), Line Charts e Correlogramas.
* **Categorias:** Gráficos de Barras e Mosaic Plots.
* **Interatividade:** Implementação de gráficos interativos para exploração dinâmica.

### 🔬 Análise Estatística
* Cálculo de estatísticas descritivas (média, mediana, desvio padrão).
* Criação de tabelas de frequência e de contingência.
* Análise de correlações entre variáveis.
